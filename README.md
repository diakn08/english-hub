# EnglishHub 🎓

## Запуск

```bash
npm install
npm run dev
```

Открыть: http://localhost:5173

## Страницы

- `/` — Главная
- `/courses` — Курсы (поиск, фильтр, сортировка)
- `/courses/:id` — Детали курса
- `/login` — Вход
- `/register` — Регистрация
- `/dashboard` — Дашборд 🔒
- `/create` — Создать курс 🔒
- `/edit/:id` — Редактировать 🔒 (владелец/admin)
- `/my-courses` — Мои курсы 🔒
- `/favorites` — Избранное 🔒
- `/profile` — Профиль 🔒
- `/admin` — Админка 🔒🔒 (только admin)
